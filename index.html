視差滾動效果

作者：Vineet Kumar

什麼是視差滾動？

視差滾動是一種網頁設計技術，其中網站背景的移動速度比前景慢。 當訪客向下滾動網站時，會產生 3D 效果，增加深度感並創造更身臨其境的瀏覽體驗。視差基於視錯覺原理。 由於人眼會將靠近我們的物體感知為比遠處的物體更大， 我們會感覺到遠處的物體移動得更慢。這種錯覺早已被應用於不同媒介的視差效果中，營造逼真的效果。 它最早應用於傳統動畫，可以追溯到迪士尼的《白雪公主與七個小矮人》，以及像《超級瑪利歐》這樣的電子遊戲。 隨著 CSS 和 HTML 的進步，視差效果後來演變為我們今天所知的網頁設計世界。

視差滾動在瀏覽器中的首次實作之一是由網頁開發者 Glutnix 於 2007 年在部落格文章中創建和分享的， 其中包括範例程式碼和使用 JavaScript 和 CSS 2 的演示，支援 Internet Explorer 6 和當時的其他瀏覽器。 然而，直到 2011 年 HTML5 和 CSS 3 編碼的引入，這項技術才在網頁設計師中流行起來。 支持者認為這是擁抱網路流動性的簡單方法。支持者將視差背景作為更好地吸引使用者並改善網站提供的整體體驗的工具。 然而，普渡大學於 2013 年發表的一項研究顯示："儘管視差滾動增強了使用者體驗的某些方面， 但它不一定改善整體使用者體驗"。


為什麼我應該使用視差圖形？

    設計師應該嘗試視差圖形有兩個主要原因：

  • 它有助於講述故事

    視差圖形非常適合講故事。好的設計就是溝通——當我們設計網站時， 我們想與訪客溝通，講述關於我們產品、服務或品牌的引人入勝的故事。 視差滾動讓我們能夠讓網站訪客沉浸在獨特的線上體驗中。 甚至可以創建一個單頁網站，訪客可以在不導航到多個頁面的情況下閱讀關於您的產品或品牌的完整故事。

  • 它有助於提高使用者參與度

    視差圖形可用於提高使用者參與度。降低網站的跳出率是網頁設計中需要注意的目標之一。 跳出率衡量的是只查看您網站一個頁面後點擊「返回」按鈕或關閉分頁的網站訪客百分比。 高跳出率會損害您的搜尋排名，因此任何可以降低該比率的行動都是值得的。 精心設計的視差頁面為使用者提供了動態體驗，往往能讓他們在頁面上停留更長時間。


如果您想為您的網站添加視差滾動效果，您應該考慮以下提示。

  • 選擇性使用

    當您在一個特定元素上使用視差滾動時，它會更加強大，而不是在所有頁面上一直使用。 在標題和標題中使用它，或者僅在您的首頁上使用。 您希望視差滾動能夠加強網站的設計，而不是分散使用者的注意力。

  • 圖片壓縮

    視差滾動使用各種媒體檔案和 CSS 形狀來創造深度感。 您不希望因為載入速度太慢的大型圖片檔案而損害網頁效能和使用者體驗。

  • 玩轉色彩

    視差滾動不僅關乎前景和背景的圖片，也關乎色彩。 您可能在背景中使用更多去飽和色調，在前景中使用更生動的顏色，以創造更大的深度感。
